### example

Works like
[AQL Helpers](https://www.arangodb.com/docs/stable/drivers/js-reference-aql.html).

TL;DR

```ts
// import { aql } from '...'

const arr = [1, 2, 3];

const query = aql`for u in ${arr}`;
```

_query returns_

```ts
{
  query: "for u in @value0",
  bindVars: { value0: [1,2,3] },
}
```
